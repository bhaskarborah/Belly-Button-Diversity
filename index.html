Bacterial properties which synthesize protein can be used with products derived from plants to produce plant based meat substitutes which taste like beef. It has been found, that bacteria present in the belly button of some human bodies are best suitable to synthesize this protein. Samples have been collected from anonymous people, and ecah have been given an id number.

In the sections below, a dashboard has been built displaying the demographic details, bar-chart, bubble-chart and gauge-chart for the sample collected. Using this dashboard, it can be determined which bacteria sample can be considered for the process of protein synthesis.
